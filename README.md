# Closira Fullstack Suite

Closira is an AI-powered agentic customer support dashboard and automation tool designed to qualify, escalate, and resolve user inquiries. It acts as an operations control center for support teams to monitor inbound leads, manage automated follow-ups, and step in for human resolution.

## Repo Structure
